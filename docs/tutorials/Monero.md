通过 I2P 连接你的 Monero 钱包到远程 Monero 节点
=================================================================

### 开始之前，请注意：我对任何 MONERO 的丢失概不负责，请始终记得备份你的密钥

在本教程中，我将向你展示如何为 I2P 配置你的 Monero 钱包。
本指南也适用于自托管节点（只需更改 IP 地址即可），
但在本教程中我将使用远程节点。
为什么要这样做？如果你想要更多隐私，想在 Monero 付款时隐藏你的 IP，
或者仅仅是因为你「能做到」——那也很酷。

> 注意：本指南适用于可从 Monero 官网
> https://www.getmonero.org/downloads/ 下载的 Monero GUI 钱包。其他钱包
> 按照本指南操作可能无法使用。


## 创建你需要的隧道
打开你的 tunnels.conf 文件，将以下内容添加到文件末尾：
```
[monero]
type = client
destination = （你需要填写你想使用的节点的 b32 地址，可以在 monero.fail 上找到节点，或使用 stormycloud 的节点，我个人使用这个）
port = 18089 （或你的节点使用的端口）
adress = 0.0.0.0 （你也可以用 127.0.0.1，但我的 i2p 节点在另一台电脑上运行，所以我用 0.0.0.0）
keys = monero.dat （你可以更改文件名，但务必保留 .dat 扩展名）
```
### 然后重启你的节点（仅适用于 Linux，不适用于 Windows，如果你用 Windows 只需重启节点即可）
```
sudo systemctl restart i2pd
```
这仅适用于 systemd 系统，但如果你用的是其他 init 系统，你应该知道如何处理这条命令。

##
## 隧道设置好后，现在来设置你的钱包
如果你还没从 https://www.getmonero.org/downloads/ 下载 Monero 钱包，现在去下载，然后回到这里。
现在你已经打开了 Monero 钱包，我们需要将它切换到高级模式。进入设置，然后点击关闭钱包
（等等，在这之前，记得备份你的钱包密钥，应该不会出问题，但以防万一——而且不管怎样你都应该备份密钥）。
现在在窗口底部应该会看到一个「更改钱包模式」的按钮，点击它并切换到高级模式，然后重新登录你的钱包。
它应该会保存你的密钥，但如果没有，请使用你的密钥重新登录。

##
## 现在你的钱包已处于高级模式，可以连接到 I2P Monero 节点了
为此，我们需要再次进入钱包设置。进入设置后，点击顶部的「节点」选项卡，然后点击「远程节点」
（可能会询问是否停止本地节点，如果弹出则选择停止下载或停止本地节点——除非你想要公共节点，但本指南是针对远程节点的，所以就选停止吧）。
然后点击「添加远程节点」，点击后需要输入你的 i2pd 节点的 IP：
如果节点与钱包在同一台电脑上运行，IP 为 127.0.0.1；如果在另一台电脑上，则为那台电脑的 IP。
端口号填写你在 tunnels.conf 中的 port 值，用户名和密码留空。

### 现在你应该可以连接到节点了。你可以点击左下角的连接按钮，或在设置中点击你添加的远程节点。
现在你的钱包应该会同步该节点上的区块链，一切就绪。
感谢阅读，祝你愉快。现在你可以更匿名地进行支付了——没人会知道你给 i2pd 捐了款。嘿，就是说说而已。
